1. Clonez ce dépôt :
   ```bash
   git clone https://github.com/doloadama/Predictions_demande_energie.git
   cd nom-du-repo
   ```

2. Installez les dépendances :
   ```bash
   pip install -r requirements.txt
   ```

3. Lancez l'application :
   ```bash
   streamlit run app.py
   ```
